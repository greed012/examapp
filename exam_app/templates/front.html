Overview of the project :

The project I say exam app is a simple application for conducting online exams (more like MCQ).A user have two options.One use is to conduct exam and other to attend the exam. So we can say that there are two modes.

  • Conduct Exam
  • Attend Exam


Conduct Exam:

This is a mode in which user can create different question papers,set multiple options and set their answers and also time limit that examinor want to allocate for the examination. A 26 digit code will be generated that he can share with his examinee or students with which they can acess Attend Exam mode to give examination.There are more fatures in this interface such as viewing the result of those who attended examination, Delete question paper,edit uestion paper their answer and other too. Note: But you need to login using Google to access this mode.



Attend Exam:

This mode is used to attend the exam. But you need have 26 digit exam code(Example: KPxO8eWjhoDafJjrfmaCGZQEDp) to attend exam. If you dont have the examination code please ask your examinor for the code. What happens in this mode is you enter your name and exam code then an exam paper will pop up . After you answer the uestion and hit submit within the allocated time then the system will check you answer with already set answer(setted by your examinor) and show you your marks. Note:You can only view your marks once but you dont have to worry, Your marks will be saved in teacher's interface.
